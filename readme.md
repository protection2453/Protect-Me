# Protect-Me

## Introduce
- 본 프로젝트는 성범죄를 예방하는 웹서비스와 성범죄를 대응하는 앱서비스로 나누어 구현하였습니다.
- 웹개발 1명, 앱개발/보안 1명, 기획/보안 1명으로 진행한 프로젝트입니다.

## Environment
- OS : Window10
- Editor : Eclipse
- Language : Html5, css, javascript, php
- DB : MariaDB
- WAS : Tomcat Apache
